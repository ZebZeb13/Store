# DeviStore

## Installation

`yarn install`

`docker-compose up`

`yarn start:dev`


## Techonologies

NodeJS (backend language)-> https://nodejs.org/

NestJS (framework) -> https://nestjs.com/

TypeORM (ORM) -> https://github.com/typeorm/typeorm

MySQL (Data base) -> https://www.mysql.com/

GraphQl (Network layer) -> https://graphql.org/

GraphQl UI - Insomnia
